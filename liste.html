Front lever

  • Type de figure : Static
  • Niveau :
  • Muscles engagés : Dorsaux, abdos, épaules, pectoraux, lombaires
  • Description : Attrapez une barre en pronation and levez votre corps de sorte à être parallèle au sol avec les jambes droites. Essayez de garder vos coudes bloqués et vos épaules en position neutre, ni retracté ni contracté.
  • Progressions :
    tuck, advance tuck
    one leg, straddle
    half lay, full

Full planche

  • Type de figure : Static
  • Niveau :
  • Muscles engagés : Dorsaux, épaules, triceps, avant-bras, lombaires
  • Description : Mettez vos mains sur le sol avec un écartement plus large que celui de vos épaules, avec vos mains légèrement vers l'extérieur, avancez votre haut du corps en avant jusqu'à ce que vos pieds décollent du sol tout en gardant les jambes serrées.
  • Progressions :
    tuck, advance tuck
    one leg
    straddle, full
